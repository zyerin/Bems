# <p align = "center">BEMS 혁신제품 Web server 개발
본 프로젝트의 시스템은 외부 원격 단말의 구내 방송망의 접속 요청을 받으면 관리자 단말과 외부 원격 단말에 동시에 인증 메세지를 송신하고, 
양측의 응답을 통해 접속 권한을 설정하며, 외부 원격 단말에서 인증서버에 의해 구내 방송망을 제어한다.
