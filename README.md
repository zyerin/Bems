# <p align = "center">BEMS 혁신제품 Web server 개발 
<p align = "center">산학 협업 프로젝트를 통해 특허를 기반으로 구현한 제품인 'BEMS'의 인증서버(Web Server) 구현

## 🔦 Introduction
본 프로젝트의 시스템은 구내 방송망의 AS Event가 발생하면 인증 서버에서 이를 인지하고 관리자 단말과 외부 원격 단말에 동시에 인증 메세지를 송신하고, 
양측의 응답을 통해 접속 권한을 설정한다. 이후 인증이 완료된(관리자에 의해 접속이 허가된) 외부 원격 단말에서 인증서버를 통해 구내 방송망을 제어할 수 있다.
